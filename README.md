使用：
网站有检测机制，访问过快会被进行人机验证

脚本这里

time.sleep(60)  # 根据实际情况调整等待时间 这里设为60，这样可以被检测
或者自己设置代理池，也可以绕过检测

需要自己注册用户，因为一个用户一天只能下载5个，签个到可以在多5个，
注册用户和密码要相同，全部放在name.txt中
这里的用户名和密码都相同，如账号和密码为：aaa,aaa;bbb,bbb;ccc,ccc
则这样填入
aaa
bbb
ccc

脚本使用是python3.11，请自行使用pip3下载对应的库
